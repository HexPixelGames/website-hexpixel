A downloadable asset pack A set of 32x32 basic sprites and tilesets for top down games. Files contain a Unity package and extracted textures for use in other engines.

FEATURES:

- 48 props
- 15 grasses
- 3 trees
- 256x256 grass tileset
- 256x256 stone ground tileset
- 512x512 wall tileset
- Seperated shadow sprites
- Simple character controller included
- Demo scene included
- Supports Standard RP, URP and HDRP
